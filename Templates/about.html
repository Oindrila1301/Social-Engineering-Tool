üîç How It Works

  1. User initiates a scan via the web interface.
  2. The application connects to the Gmail inbox using OAuth authentication.
  3. Emails are fetched and links are extracted from the content.
  4. Each link is analyzed using:
    • ML-based Phishing Detection
    • Fuzzy Logic Scoring (based on URL length and risk score)
  5. Final results are shown using pie and bar charts, along with detailed breakdowns.
